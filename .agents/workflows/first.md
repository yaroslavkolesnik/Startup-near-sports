---
description: Generates a complete fullstack feature for Django and React Native.
---

When triggered, execute the following steps sequentially:

1. Create or update models.py in backend with the new model.
2. Register the new model in admin.py in backend.
3. Create or update serializers.py in backend.
4. Create or update views.py in backend.
5. Register the new endpoint in backend urls.py.
6. Create an API service file in frontend to call the new endpoint.
